## Forbidden

Do not add source images, print files, PDFs, card images, prompts, planning documents, logs, private notes, email addresses, phone numbers, workplace names, addresses, or any personal information.

Do not add unreviewed videos, raw downloads, audio, intermediate thumbnails, local paths, or files with private story/IP names.

## Git

Do not use `git add .`.
Add only allowlisted paths explicitly.
